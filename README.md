Landslide E-Portal

A full-stack disaster management and reporting platform designed to help users report landslides, analyze risk-prone areas, and improve awareness through an interactive web interface.

Features
Landslide reporting system
Interactive and responsive frontend
Location-based risk checking
AI chatbot assistance
Real-time user-friendly interface
Multiple pages with smooth navigation
MongoDB database integration
Firebase hosted frontend
Technologies Used
HTML
CSS
JavaScript
Node.js
Express.js
MongoDB
Firebase Hosting

Installation

Clone the repository:

git clone https://github.com/yourusername/Landslide-E-portal.git

Open project folder:

cd Landslide-E-portal

Install dependencies:

npm install

Run server:

node server.js
